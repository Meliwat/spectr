Workshop Notes

Obsidian's canvas is a deep charcoal #1E1E1E — a developer-tool workshop, not a glossy app. Obsidian Purple is the single connective accent: #7C3AED for actions, #A78BFA for [[wikilinks]], active graph nodes, and selection. Two type systems exist on purpose — Inter for chrome and reading, JetBrains Mono for the source editor and command palette — and markdown syntax markers stay dimmed-but-visible in #6B6B6B so you always know you're editing text. Elevation is a charcoal step plus a 1pt #363636 divider; real shadows appear only on the floating command palette. The graph view and backlinks pane are the twin pillars of the knowledge system.

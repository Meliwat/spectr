Dark Mode & HUD Notes

Zwift is effectively always-dark — the screen is dominated by a full-bleed 3D virtual world, so all chrome is glassmorphic: translucent dark panels (rgba(12,12,12,0.6)) with blur(8px) so the road and riders read through. Zwift Orange #FC6719 is the brand action color (Start Ride, the rider jersey, route progress, active power-up). Live ride metrics use a fixed semantic mapping — Power orange #FC6719, Cadence yellow #E8C547, Heart-rate red #F0413E, W/kg cyan #2BD4D9 — in big tabular Barlow numerals so they're legible from a turbo trainer two metres away. App-shell screens (menus, route picker) sit on a solid #161616 base.

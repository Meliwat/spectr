Dark Mode Notes

ESPN's dark mode is a near-black #0E0F11 (not pure black) so the scores ticker and game cards lift cleanly on #212327 surfaces. ESPN Red #D50A0A is the single brand accent — the logo block, the primary "Watch Live" CTA, the active-tab indicator dot, and section accents. Live state is a brighter red #FF1A1A with a pulsing dot; finals dim to gray; the winning team's score turns green #1FAA59. Scores always use tabular numerals so columns stay aligned across every card.

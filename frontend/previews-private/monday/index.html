Dark Mode Notes

monday.com's night mode uses a deep indigo canvas #181B34 — distinctly blue-tinted, never neutral black. Status cells stay fully saturated and full-bleed in both themes: green #00C875, orange #FDAB3D, red #E2445C — they carry all of the board's meaning, so they never dim. The leading group-color stripe (6pt) keeps items visually bucketed. Primary blue #0073EA drives the FAB and CTAs. Text inverts to near-white #F5F6F8; surfaces lift from the canvas in cool indigo steps (#20243F → #292F4C).

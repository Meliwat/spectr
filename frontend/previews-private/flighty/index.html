Dark Mode Notes

Flighty's canvas is #0B0B0F — a deep instrument-panel black, never pure #000000 (the slight lift keeps depth in the map). Flighty Blue #0A84FF is the single accent: primary action, the live great-circle arc, and active states. Status colors are strict and never decorative — green #30D158 = on-time, amber #FFD60A = delayed, red #FF453A = cancelled, on chips only. All flight data uses tabular figures so live updates never jitter. The only colored light is the soft blue glow behind the arc and plane marker.

Dark Mode Notes

Tubi is dark-only — there is no light mode. The canvas is a deep indigo-black #0A0A2A, not neutral grey, which makes the purple→magenta gradient (#7408FF → #FF00FF) glow. Cards lift to #15153D. The primary play button is pure white #FFFFFF with near-black text for maximum contrast against the cool field; the gradient is reserved for the wordmark, secondary CTAs, the active tab, and progress bars. The yellow #FFD400 "FREE" tag is the one warm accent — it signals Tubi's core promise. Posters render full-color; only chrome dims.

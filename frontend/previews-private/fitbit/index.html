Dark Mode Notes

Fitbit's dark mode is a deep teal-black, not neutral — the canvas #001017 and surface tiles #002A3A carry the brand's cyan undertone. Fitbit Teal #00B0B9 shifts to a brighter #21D9CE for text/links on dark so it stays legible. Each health metric keeps its dedicated hue across themes — heart-rate coral #FF6B81, sleep purple #7C5CFF, readiness lime #B8E986, zone orange #FF8A3D, calories gold #FFC233 — so a glance reads the same data anywhere. Progress rings keep their metric color on a muted #0A475E track. The light theme swaps canvas to #FFFFFF and text to #0B2A33 while every metric hue stays identical.

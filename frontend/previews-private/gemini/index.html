Explain what a transformer is, simply.
A transformer is a neural network architecture that reads an entire sequence at once and learns which parts matter most using a mechanism called attention. Instead of processing words one by one, it weighs every word against every other word in parallel — which is why it scales so well to long context.

Light + Dark Notes

Gemini reads as a document, not a feed. There are no cards around turns — the user turn is a soft #F0F4F9 chip (trailing-aligned, 4pt pointing corner) and the assistant turn is plain markdown text on the canvas, led by a gradient sparkle. Dark mode inverts canvas to #1E1E1E and surface to #282A2C; links/send use #8AB4F8 for AA contrast. The gradient stops (#4285F4 → #9B72CB → #D96570) stay identical across themes — it is brand, never a body-text tint or button fill.

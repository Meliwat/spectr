Dark Mode Notes

Postman's dark mode is a neutral grey, canvas #1A1A1A with cards lifting to #212121. Postman Orange #FF6C37 is identical across light and dark — it's reserved exclusively for the Send button, active tab underline, and primary CTAs, never for decoration. The real color system is the HTTP-method palette: GET #6BDD9A, POST #FFE47A, PUT #74AEF6, PATCH #C0A8E1, DELETE #F79090 — these soften on dark vs the saturated light-mode versions but keep their hue identity. All request/response data is JetBrains Mono with JSON syntax coloring.

Dark Mode Notes

Jira's dark theme is built on Atlassian's neutral ramp — canvas #1D2125, cards elevate to #22272B, columns and inputs to #2C333A. The signature Jira Blue #0052CC stays for brand moments, but interactive blue brightens to #1868DB so links and the active tab clear the dark canvas at WCAG AA. Status lozenges keep their semantic hue families (blue / green / red / neutral) with darkened fills and bright text. Text never goes pure white — primary is #C7D1DB, subtle #9FADBC, echoing the warm-neutral #172B4D ink of light mode inverted.

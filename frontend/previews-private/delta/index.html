Boarding Pass Anatomy

The digital boarding pass is Delta's hero surface: a navy→red top strip, passenger name in caps, a three-cell Gate / Seat / Group grid, a SKY PRIORITY zone band, and a scannable barcode block at the foot. It appears in Apple Wallet and in-app; the in-app version adds a live Flight Status timeline above it.

Dark Mode Notes

Delta's dark mode keeps the brand navy alive: the canvas is a deep navy-charcoal #0B1320 (not neutral black) so the boarding-pass gradient header (Delta Blue #003268 → red #C8102E) still reads as Delta. The red widget accent #C8102E is reserved for the primary CTA only. SkyMiles gold #C99700 marks Medallion status. Status colors are functional and constant: on-time green #1E8E5A, delayed gold #C99700, canceled red #E0414E. The scannable barcode inverts to a light bar pattern so a gate scanner still reads it on a dimmed screen.

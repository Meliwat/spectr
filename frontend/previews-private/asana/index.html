Dark Mode Notes

Asana's dark mode uses a near-neutral charcoal canvas #1E1F21 with surfaces lifting in tight 7-point steps (#252628 → #2E2F31). The coral accent #F06A6A is identical across themes — it stays warm and legible against dark. The multicolor project palette (plum #4573D2, aqua #4ECBC4, green #62D26F, yellow #F8DF72) is rendered as low-opacity tint chips (16-18%) so status pills and project tags glow softly without overwhelming the list. Text inverts to a warm off-white #F5F4F2; checkmark fills stay full-saturation green.

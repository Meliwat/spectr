Light Mode Notes

Dropbox is a warm-white workspace. Canvas is true #FFFFFF; grouped sections lift to warm cream #F7F5F2 with a warm hairline #E6E1DA. Text is warm ink #1E1919, never pure black. Dropbox Blue #0061FF is rationed to primary actions, the active tab, links, and selection. File-type icon colors (PDF red, image teal, sheet green) are illustration, not UI accent. The dark theme inverts to a warm charcoal #1E1919 canvas with blue lightened to #3D8BFF for AA contrast.

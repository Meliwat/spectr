Dark Mode Notes

Match's dark mode is a warm near-black, not pure black. Canvas sits at #141414 with a faint red undertone; the profile card itself is full-bleed photography so it never dims. Match Red shifts from #E92434 to the brighter #F0203E on the circular Like button and active tab so it stays vivid against dark imagery. The action dock circles use #1E1E1E with hairline #322C2C borders instead of heavy fills. The "It's a Match!" celebration keeps a deep maroon-to-ink gradient so the red wordmark glows.

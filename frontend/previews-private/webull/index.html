Dark Mode Notes

Webull is dark-native — there is no meaningful light mode for a trading terminal. The canvas is a near-black #0B0E11 so candlesticks, depth bars, and the blue→cyan brand gradient pop with maximum contrast. Surfaces lift only slightly to #14181D and #1C2127; chart grid lines are an almost-invisible #1A1F26. Unlike a calm brokerage, performance color is loud and pervasive: up is #00C076, down is #FA5252, applied to candles, sparklines, % chips, the order ladder, and the split Buy (green) / Sell (red) action pair. The brand gradient #1B9EFB → #20D5C4 is reserved for identity, the primary Trade action, selected timeframes, and the paper-trading badge. All numerals are tabular so price ladders and option chains align to the decimal.

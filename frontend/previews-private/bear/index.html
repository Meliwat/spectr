Dark Mode Notes

Bear ships several themes; the default dark theme ("Charcoal") sits on #21252B — a warm blue-charcoal, never pure black. Cards and the tag sidebar lift to #282C34. The red→orange gradient (#E0566F → #FF8A65) is the single accent and is identical across light and dark — it reads beautifully on both. Body text inverts to #E8EAED; inline #hashtags stay red, links stay #5AAFEF. Markdown syntax markers dim to #6B7280 so prose stays the focus. Code blocks darken to #282C34 with JetBrains Mono.

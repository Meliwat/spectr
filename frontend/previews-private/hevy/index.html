Dark Mode Notes

Hevy is dark-first — the app launches into a near-black training canvas #0E1116 so the gym screen never glares. Cards lift to #161B22; editable set cells sit on #1F2630. Hevy Blue #1E6FFF is the only action color — finish button, active tab, exercise titles. Completed sets tint green #2FBF71 at 16% and a gold #F5B83D PR badge fires when you beat a record. All numerics use tabular figures so the set table never shifts as you type.

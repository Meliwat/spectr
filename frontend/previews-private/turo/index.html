Dark Mode Notes

Turo's dark theme is near-black #0F0F12 so full-bleed car photography pops without competing chrome. Turo Purple #593BFB stays the booking action color on both themes; on dark it can brighten to #7C5CFF for inline links to clear WCAG AA. Turo Teal #5CE0B8 is the trust/value accent — All-Star Host badges, savings, instant-book — and reads beautifully against the dark surfaces. Photos are never dimmed; only UI chrome darkens. The sticky price + Book bar floats on a blurred near-black panel so it never fights the imagery above it.

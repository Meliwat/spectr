Dark Mode Notes

Deezer is dark-native — there is no real light theme. The canvas is a violet-tinted near-black #0F0D13, not pure black, so the purple→pink Flow gradient feels embedded in the same color world. The gradient (#A238FF → #FF0092) is treated as a scarce resource: it appears on the 68pt circular play button, the scrubber fill, the Flow badge, and the now-playing equalizer marker — nowhere else. Album/Flow artwork stays full-color and luminous; UI chrome (chevrons, secondary transport, tab icons) stays monochrome #A29CB0 so the eye is always pulled to where the music is playing.

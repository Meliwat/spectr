Light Mode Notes

Klarna is soft, friendly, and bright with a single dark anchor — the black payment-summary hero at #0B051D. Klarna Pink #FFB3C7 is the only action color and always carries near-black text, never white. The Pay-in-4 schedule timeline — four dated dots on a connecting track, the first filled pink — is the signature object: the plan is never hidden behind a summary. Radii run notably soft (28pt CTA, 20–24pt cards) and transitions glide unhurriedly — the softness is the brand.

Dark Mode Notes

Linear's canvas is #08090A — darker than the common #121212 dark mode, almost OLED true-black, so the issue list becomes the entire surface. Elevation is a single background step (#08090A → #1C1D1F) plus a 1pt #23252A border — real shadows appear only on the floating command menu. Linear Purple #5E6AD2 is the only accent: primary action, the focused command row, and the active selection. Status is a drawn icon language, never a colored text badge.

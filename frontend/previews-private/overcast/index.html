Dark Mode Notes

Overcast ships a warm paper-cream light theme (#FBFAF6 canvas, #F7F5EF surfaces) and a true dark theme (#121212 canvas, #1C1C1E surfaces). Overcast Orange #FC7E0F is the single accent and is identical across themes — it rings the circular play control, colors the scrubber, marks active Smart Speed / Voice Boost toggles, and tints links and the tab selection. The UI is deliberately restrained: one accent, generous list spacing, no gradients in chrome. Cover artwork stays full-color; everything else is monochrome so the orange reads instantly.

Dark Mode Notes

Skype's dark mode uses a neutral near-black (#16161A) so the cloud-blue brand pops. The bright Skype Blue #00AFF0 drives buttons, call FABs, and tab selection across both themes; outgoing bubbles use the deeper, more readable #0078D4. Incoming bubbles invert from light #EBEBEF to #2A2A30. Accept-call green #2DC26B and end-call red #E8364F are constant. Video tiles keep full-color gradients; chrome dims, brand blue does not.

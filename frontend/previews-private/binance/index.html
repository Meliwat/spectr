Dark Mode Notes

Binance is dark-first — the trading UI was designed for the deep canvas #0B0E11; cards lift to #181A20 and #1E2026. Binance Yellow #F0B90B is the single brand accent and stays identical across the (rarely used) light theme. Up/down semantics never invert: green #0ECB81 and red #F6465D are absolute. All numeric data — prices, balances, order book, P&L — is set in IBM Plex Mono with tabular figures so columns align to the pixel. The light theme simply swaps canvas to #FFFFFF and text to #1E2329; accents and semantics are unchanged.

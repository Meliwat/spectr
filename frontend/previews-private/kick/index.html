Dark Mode Notes

Kick is dark-only — a near-black #0E0E10 canvas, not pure black, so the green doesn't vibrate against true #000. Kick Green #53FC18 is an intentionally loud, almost-radioactive lime: it owns the logo, the Follow/Subscribe buttons, the active tab, online avatar rings, and links — and almost nothing else, so it stays a shock of color in an otherwise monochrome UI. Chat is the second pillar: usernames are color-coded by role (mod #3EA6FF, sub #FFC700, VIP #FF4FD8, default randomized green-ish), each prefixed by a tiny badge chip, with custom emote sprites inline. The live indicator is always #FF1F44. Stream video stays full-color; chrome and chat scaffolding stay grey.

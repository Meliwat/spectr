Dark Mode Notes

Kik's dark mode uses a neutral near-black (#121316) so the modern Kik Blue stays vivid. The brand Blue #00B0F0 drives the outgoing bubble (with dark-cyan ink #002A36 for AA), buttons, and tab selection across both themes; incoming bubbles invert from light #E9EAEC to #25282D. The legacy Kik Green #82BC23 survives only as a heritage accent. The single-letter S/D/R receipt turns Blue at "R" (Read). Chrome dims; brand Blue does not.

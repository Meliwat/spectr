Board Backdrop Notes

Trello has no fixed canvas — the board background (solid #0079BF, a gradient, or a photo) owns the screen, and pale #F1F2F4 lists of white cards float on it. There is no bottom tab bar: the board scrolls horizontally and the top bar holds navigation. Card text stays navy #172B4D for legibility over any backdrop; the six-color label palette is the one place color is free, because there it is data. Shadows use the navy tint rgba(9,30,66,…) so depth reads on both color backdrops and pale lists — and a card's shadow jumps dramatically the instant you pick it up.

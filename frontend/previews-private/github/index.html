Dark Mode Notes

This is GitHub's Primer "dark default" theme. The canvas is #0D1117 with surfaces stepping up through #161B22 (subtle) and #21262D (buttons), bordered by #30363D. Hyperlinks and active elements use accent #2F81F7; the famous green #238636 is reserved for the "Code" and "Merge pull request" buttons. State is color-coded semantically: open = green #3FB950, merged = purple #A371F7, closed = red #F85149, draft = grey. The contributions heatmap runs #0E4429 → #006D32 → #26A641 → #39D353 — GitHub's most recognizable visual.

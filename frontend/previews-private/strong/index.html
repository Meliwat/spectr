Dark Mode Notes

Strong is famously dark-first — a flat, near-black canvas #1A1A1A with almost no chrome. Cards lift subtly to #242424; editable set cells sit on #2E2E2E. Strong Blue #2F80ED is the single accent — Finish button, active tab, exercise titles, the rest-timer fill. Logged sets fill green #27AE60 at 18% and an amber #F2C94C PR flag appears when you set a record. Every figure is tabular so the set table never reflows mid-workout.

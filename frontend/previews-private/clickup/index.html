Dark Mode Notes

ClickUp's dark mode uses a deep blue-violet canvas #1B1B2E that complements the brand gradient. The signature 3-stop gradient (purple #7B68EE → pink #FD71AF → blue #49CCF9) is reserved for the FAB, primary CTA, and AI surfaces — it is the brand's hero moment and stays identical across themes. Custom status pills stay saturated (they group the list and carry meaning); priority is a colored flag triangle (Urgent #F44E6E / High #FF9F1A / Normal #49CCF9 / Low gray). The UI is intentionally dense — many columns of metadata per task — so surfaces lift in tight steps (#232338 → #2C2C44) and text inverts to near-white #F1F1F6.

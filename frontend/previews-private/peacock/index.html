Dark Mode Notes

Peacock is dark-only — there is no light theme. The canvas is a deep indigo-black #0A0A14 that warms toward violet #140E26 beneath hero artwork, so the billboard glows rather than sits on flat black. The five-color feather gradient (yellow → orange → pink → purple → blue) is reserved for the logomark, premium upsell, and selection accents — never as a fill behind text. The primary "Play" CTA is always solid white #FFFFFF with a near-black label for maximum contrast over any poster. Live and sports surfaces use NBC red #E5142B with a pulsing dot. Poster art is full-color and full-bleed; the surrounding chrome stays muted so the catalog is the hero.

Boarding Pass & Seat Map Anatomy

United's signature flow is flight detail → boarding pass. The pass has a solid United Blue #002244 header with the globe mark and a gold PREMIER · GROUP 1 band, a Gate / Seat / Boards grid, a perforated dashed divider with notch cut-outs, and a scannable barcode. The interactive seat map uses Rhapsody Blue #1414FF for the selected seat and a translucent blue for Economy Plus rows.

Dark Mode Notes

United's dark mode keeps the brand navy alive: the canvas is a deep navy #0A1424 (not neutral black) so the United Blue #002244 pass header and flight-detail gradient still read as United. Rhapsody Blue #1414FF is the electric primary-action accent — buttons and the selected seat. Premier gold #C2A14D marks MileagePlus elite status only. Functional status colors are constant: on-time green #1E8E5A, delayed amber #C98A1E, canceled red #D8434F. The barcode inverts to a light bar pattern so a gate scanner reads a dimmed screen.

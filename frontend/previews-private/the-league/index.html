Dark Mode Notes

The League is dark-native — it has no light mode in spirit. The canvas is a near-true-black #0A0A0A evoking a private club after dark; surfaces lift only to #141414. Gold #C8A35A is rationed: the crest, hairline rules, the single primary CTA, and the credential bullets — nothing else. There are no drop shadows; depth is created entirely by 1px gold-tinted hairline borders (#2A2620) and an embossed calling-card treatment. The display serif (Cormorant Garamond) carries the editorial, members-club gravitas; the sans (Jost) handles credentials and UI in a quiet uppercase.

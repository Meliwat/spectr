Dark Mode Notes

Midjourney is dark-only and the canvas is true black #000000 — not a warm charcoal — because OLED black makes generated imagery glow. There is intentionally no brand accent: white #FFFFFF is the only "primary" (Imagine button, active tab, upscale chips). All chrome — prompt placeholder, job metadata, icons — sits at #A1A1A1 or dimmer so it never competes with the 2×2 grid. Prompts and parameters render in monospace; the colored 4-up grid is the single source of saturation in the entire UI.

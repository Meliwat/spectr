Dark Mode Notes

iHeartRadio's dark mode is a warm maroon-black #120A0E — a red-tinted dark, never pure black — so the iHeart red system feels native to the canvas. The brand system (#C6002B red → #E40A5D magenta, with coral #F23A2F reserved for live signals) is scarce: it lights the circular play button, the heart logomark, the pulsing LIVE badge, and the indeterminate scanning bar. Because live radio cannot seek, there is no scrubber — the scanning bar conveys "streaming now" instead of progress. Chrome (chevrons, secondary transport, tab labels) stays monochrome #B8A0A8 so the eye is pulled to the live station and its play control.

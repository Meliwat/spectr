Dark Mode Notes

Marriott Bonvoy's dark mode keeps the premium, restrained feel: canvas at #121214, surfaces lifting to #1A1A1D. Bonvoy Navy #16264A stays the panel/CTA-secondary color across themes; gold #B3852A brightens to #D2A23E for points balances, tier eyebrows, and the gold "Book" CTA so member value still glows on dark. Gold is disciplined — it marks loyalty value (points, tiers, redemption), never decoration. Hotel photography stays full-color; only chrome dims.
